Computational Efficiency: Classic R-CNN vs. Fast R-CNN

Classic R-CNN / Sliding Window

Crop every proposal from image, resize it, and run the entire backbone network separately.

FLOPs = N_{proposals} \times FLOPs_{Backbone} - GigaFLOPs
-x Faster

Fast R-CNN (Shared Features)

Extract features for full image once. Project proposals, RoI pool, and run lightweight sibling heads.

FLOPs = FLOPs_{Backbone} + N_{proposals} \times FLOPs_{Heads} - GigaFLOPs
