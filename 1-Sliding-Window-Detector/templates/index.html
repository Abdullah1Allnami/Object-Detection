How does it work? (Roadmap Concept Explainer)

1. The Classifier

Standard CNN classifiers (like our Custom Digit ResNet50 or pre-trained ResNet50) only answer: "What is the single main object in this cropped square?". They cannot localize multiple objects or detect empty space natively.

2. The Sliding Window

To detect *where* objects are in a larger scene, we slide a bounding box over the entire image step-by-step. Each cropped window is sent to the classifier. Detections exceeding our Confidence Threshold are flagged.

3. Non-Maximum Suppression (NMS)

Since stride is small, a single object yields many overlapping hits. **NMS** calculates overlap (IoU) and retains only the box with the highest classification score, suppressing duplicate overlapping predictions. This is step 3 in the roadmap!
